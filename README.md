# NodeFarm
This is a simple E-commmerce website built using wiht Node.js for backend and vanilla JavaScript for the frontend. It allows users to browse through a list of products, add them to a shopping cart, and place an order.

Project Structure
server.js: Entry point for the application
routes/: Contains all the routes for the application
controllers/: Contains all the controllers for the application
models/: Contains all the models for the application
views/: Contains all the views for the application
public/: Contains all the static assets for the application

Features
Browse through a list of products
Add products to a shopping cart
Place an order
View past orders

Improvements
